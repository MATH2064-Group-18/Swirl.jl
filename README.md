# Swirl.jl

Welcome! Swirl.jl is a Julia package for fluid simulation.

## Getting Started

To use simply add the package
```julia
Pkg.add("https://github.com/MATH2064-Group-18/Swirl.jl")
```
Or in the Julia REPL, type `]` and then run:
```julia
pkg> add "https://github.com/MATH2064-Group-18/Swirl.jl"
```
